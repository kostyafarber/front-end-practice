### What I learned
I learnt a lot about using the grid layout in this one.

```css
.stats {
    display: grid;
    grid-template-columns: 1fr 1fr 1fr;
    grid-template-rows: 1fr 1fr;
    margin-top: 7.6rem;
    grid-row-gap: 0.3rem;
}
```
